# Work Day Scheduler
--------------------

## Description
---------------
- This project helps employee plan out their work day
- It will display standard business hours (e.g. 9am-5pm)
- Planner is color coded to label past, present, and future events
- Saved events persist

## Technologies
----------------
- Bootstrap 4
- Jquery/Jquery UI
- Font-Awesome
- Google Fonts
- Moment.js
